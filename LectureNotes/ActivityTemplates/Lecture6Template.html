Accounting for Trends

Part 1

  1. Download the data time_series.csv from Canvas and create a sub-folder in your Lecture6 folder called Data. Read in the data. Extract y1, the first time series, and name it x_t.
library(astsa)
time_series <- read.csv("../Data/time_series.csv")
x_t <- time_series$y1
  1. Plot the data
tsplot(x_t)
  1. In separate plot, again plot the time series and also plot a moving average smoother over the time series (use any \(p\) you’d like).
tsplot(x_t)

ma_x_t <- stats::filter(x_t, filter =rep(1/5,5))

lines(ma_x_t, col = "blueviolet")
  1. Detrend the time series with respect to the moving average estimate and plot the de-trended time series.
detrended_ma <- x_t - ma_x_t
tsplot(detrended_ma)
  1. In a separate plot, again plot the time series and also plot a simple linear regression line using 1:length(x) where x is the time series you are analyzing.
time_index <- 1:length(x_t)
lm_x_t <- lm(x_t~time_index)
tsplot(x_t)
reg_x_t <- lm_x_t$fitted.values
lines(reg_x_t, col = "turquoise", lwd = 2)
  1. Detrend the time series with respect to the regression and plot the detrended series.
detrended_reg <- x_t - reg_x_t
tsplot(detrended_reg)
  1. Difference the time series and plot it in yet another plot.
differenced <- c(diff(x_t), NA)
tsplot(differenced)
  1. Run par(mfrow = c(2,3)) and then re-run the plotting code for all 6 plots you just made in steps 1-6.
par(mfrow = c(2,3))

tsplot(x_t)

# number 3
detrended_ma <- x_t - ma_x_t
tsplot(detrended_ma)
  
# Number 4

tsplot(x_t)
reg_x_t <- lm_x_t$fitted.values

# number 5

# Number 6
